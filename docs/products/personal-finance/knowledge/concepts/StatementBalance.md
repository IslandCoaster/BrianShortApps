# Statement Balance

## Definition

The historical balance that exists at the statement closing date.

Statement Balance is immutable.

Once a statement closes, this value never changes.

---

## Owner

Obligation State Engine

---

## Used By

Account State Engine

Credit Position Engine

Debt Position Engine

Recommendation Engine

---

## Notes

Payments made after statement closing do not change Statement Balance.

Payments reduce Remaining Obligation.

Projected Statement Balance represents a different concept.
