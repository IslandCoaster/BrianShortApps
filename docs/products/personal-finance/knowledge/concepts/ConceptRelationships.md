# Concept Relationships

## Purpose

Define how core financial concepts relate to one another.

Concept relationships help prevent engines from confusing similar financial terms.

## Product Principle

No engine may invent a concept.

If an engine requires a financial concept, that concept must exist in the Financial Knowledge Base first.
