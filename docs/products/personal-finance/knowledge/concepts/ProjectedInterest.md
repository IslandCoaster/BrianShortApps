# Projected Interest

## Definition

The estimated interest expected to post on the next statement if no additional financial activity occurs.

---

## Characteristics

Forecast

Mutable

Scenario-dependent

---

## Owner

Interest Engine

---

## Inputs

- Daily Ledger
- Interest Rule Set
- Grace Period State
- Statement Period

---

## Product Principle

Projected Interest is never entered manually.

It is always calculated.

Projected Interest changes whenever financial events are replayed.
