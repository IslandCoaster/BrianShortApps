# Resource Naming

## Purpose

Resource names must communicate scope, owner, and purpose.

## Platform Prefix

Platform resources use:

bsa-*

Examples:

- bsa-platform-console
- bsa-registry-api
- bsa-root-diagnostics

## Application Prefixes

Application resources use application-specific prefixes.

Examples:

- mco-*
- flightlog-*
- debt-*

## Applies To

This convention applies across:

- Cognito
- API Gateway
- Lambda
- DynamoDB
- S3
- IAM
- CloudWatch
- EventBridge
- Secrets
- Future infrastructure
