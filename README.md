# BrianShortApps

BrianShortApps is a software platform for building, deploying, and operating multiple independent applications on a shared engineering foundation.

## Applications

- MCO Satellite
- Flight Log
- Debt Planner

## Platform

The platform provides:

- Identity
- Registries
- Platform Console
- Diagnostics
- Infrastructure
- Shared Services
- Deployment

Applications build upon the platform while remaining independently deployable and independently evolvable.
