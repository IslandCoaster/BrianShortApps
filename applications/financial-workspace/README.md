# Financial Workspace

Financial Workspace is the first private BrianShortApps application.

It provides deterministic personal financial management built on an operational ledger.

Current components:

- Domain Model (`packages/finance`)
- Infrastructure (`applications/financial-workspace/infrastructure`)
- Web Application (`applications/financial-workspace/web`)

The application begins from a user's current financial position and derives projections, liquidity, funding, and settlement behavior from the operational ledger.
