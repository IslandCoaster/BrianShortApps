# @bsa/design-system

## Purpose

The Design System provides the shared visual language for every BrianShortApps application.

It defines reusable UI components, design tokens, layout primitives, typography, spacing, icons, themes, and interaction patterns.

## Consumers

- Platform Console
- Mission Control
- Developer Portal
- MCO Satellite
- Flight Log
- Debt Planner
